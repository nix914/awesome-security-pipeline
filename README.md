## üîß Features
The awesome-security-pipeline includes a variety of features designed to enhance your security efforts in development:

- **Secrets Detection**: Quickly identify sensitive data in your code.
  
- **Software Bill of Materials (SBOM)**: Get a complete inventory of the components that make up your software, which helps in transparency and risk management.

- **Static Application Security Testing (SAST)**: Scan your code for vulnerabilities early in the development process.

- **Software Composition Analysis (SCA)**: Evaluate open-source components for known vulnerabilities.

- **Infrastructure as Code (IaC) Security**: Ensure that your infrastructure files follow security best practices.

- **Container Scanning**: Detect vulnerabilities in containerized applications before deployment.

- **Kubernetes Security**: Manage and secure your Kubernetes clusters efficiently.
